# Question Classification using Machine Learning

## Overview

This project focuses on classifying questions based on their content using machine learning techniques. It applies natural language processing steps to convert text into numerical features and trains models to make accurate predictions.

## What I did

* Cleaned and preprocessed text data
* Converted text into features (TF-IDF / vectorization)
* Trained machine learning models
* Evaluated model performance

## How it works

The notebook walks through the full pipeline: data preprocessing, feature extraction, model training, and evaluation. It shows how raw text can be transformed into meaningful predictions.

## Notes

This project helped me understand how machine learning can be applied to real-world NLP problems and how different steps affect model performance.
